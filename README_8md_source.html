1 # BeepSpeech
2 
3 BeepSpeech is intended to be able to create sounds for scrolling text. It has
4 the capability to create beep-speech similar to that of animal crossing's
5 animalese.
6 
7 To do this, there are several provided classes:
8 
9 * beepspeech::ISound audio file interface.
10 * beepspeech::Sounds beepspeech::ISound container
11 * beepspeech::Voice Sound Player
12 * beepspeech::phonetics::Parser Converter from text to phonetic alphabet
13 
14 There are several possibilities available with BeepSpeech:
15 
16 ## Simple beeps
17 
18 By providing one sound object, you can make simple speech. Assigning this beep
19 with a letter such as `b`. And creating a
20 beepspeech::phonetics::RegexRule('\\w', 'b'). All text provided can make beep
21 sounds.
22 
23 ## Scripted sounds
24 
25 You could go one step further, and add multiple sounds, each assigned to a
26 different letter. These sounds can be used to make listening a bit more bearable.
27 
28 ## Animalese like speech
29 
30 Creating a speech like beep is the most advanced use for this library. You could
31 use the phonetics::Parser to convert text to the used sounds, the resulting list
32 can be fed into a Voice to play that speech.
33 
34 > Do note that if many rules are made in beepspeech::phonetics::Parser, it can
35 > make parsing slow, especially for long text as finding the correct rule will
36 > be `O log(n^2)`
